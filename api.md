# API ENDPOINTS

| Endpoint                              | Описание                              |
| ------------------------------------- | ------------------------------------- |
| /ws                                   | Открытие websocket соединения         |
| `POST` /users                         | Создание нового пользователя          |
| `GET` /users                          | Получение всех пользователей          |
| `PATCH` /users                        | Изменение данных о пользователе       |
